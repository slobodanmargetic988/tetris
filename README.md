# Tetris

Canary target project for OP Simple autonomous builder runs.

## Run on localhost

From the repository root, run:

```bash
npm install
npm run dev
```

When Vite prints a local URL (normally `http://localhost:5173/`), open it in your browser to play.

Normal player flow starts immediately on the rendered board; no debug-only controls are required.

## Production-style local check

```bash
npm run build
npm run preview
```

Then open the preview URL printed in the terminal.
